# Clase2.API_Licencias

## Descripción

API ASP.NET Core que expone operaciones para gestionar licencias.

## Funcionalidad

- Registrar una licencia.
- Obtener el listado de licencias registradas.

## Endpoints

- `POST /api/licencias`
- `GET /api/licencias`

## Swagger

Incluye Swagger UI para probar endpoints durante desarrollo.

Ruta habitual:

- `https://localhost:{puerto}/swagger`

## Dependencias principales

- `Swashbuckle.AspNetCore`
- `Clase2.Servicio`
