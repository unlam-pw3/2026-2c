# Clase2.FrontForm

## Descripción

Aplicación WinForms para interactuar con el sistema de licencias desde interfaz gráfica.

## Funcionalidad

- Cargar datos de licencia desde formulario.
- Registrar licencias.
- Consultar licencias y mostrarlas en grilla.

## Integración

El front consume la API `Clase2.API_Licencias` por HTTP/HTTPS.
Para funcionar correctamente, la API debe estar iniciada antes que este proyecto.
